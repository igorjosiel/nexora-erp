# Nexora ERP

🇧🇷 **Português** | 🇺🇸 [English](README.md)

## Sobre o projeto

O **Nexora ERP** é uma plataforma SaaS de gestão empresarial desenvolvida com foco em arquitetura de software, boas práticas de desenvolvimento e tecnologias modernas.

O projeto está sendo construído como um sistema de longo prazo para simular um ambiente de produção real, permitindo a prática de desenvolvimento full stack, modelagem de banco de dados, testes automatizados, documentação e DevOps.

Mais do que um simples CRUD, o objetivo é desenvolver um sistema escalável, organizado e de fácil manutenção.

---

## Objetivos

* Desenvolver um ERP moderno e escalável.
* Aplicar princípios de arquitetura de software.
* Praticar boas práticas de desenvolvimento.
* Aprofundar conhecimentos em Node.js, TypeScript, Sequelize e Next.js.
* Criar um projeto de portfólio que represente um sistema utilizado em ambientes reais.
