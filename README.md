# Nexora ERP

🇺🇸 English | 🇧🇷 [Português](README.pt-BR.md)

Nexora ERP is a full-stack SaaS enterprise management platform built to simulate a real-world production system.

This project is being developed as a long-term portfolio project to practice software architecture, backend development, frontend development, database design, testing, and DevOps concepts.

## Goals

- Build a scalable ERP system
- Apply software engineering best practices
- Practice clean architecture principles
- Improve backend and frontend skills
- Create a professional portfolio project

## Tech Stack

### Backend

- Node.js
- TypeScript
- Express
- Sequelize
- PostgreSQL
- JWT
- Docker

### Frontend

- Next.js
- TypeScript
- React
- Tailwind CSS
- React Query
- React Hook Form
